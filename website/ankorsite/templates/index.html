Ankor

App development framework.
Synchronize your view model across multiple platforms.

  •  JavaFX
  •  HTML5
  •  .NET
  •  iOS
  •  Android
  •  Java/JEE
Get started today

JavaFX

  • Implement your model server side in Java and connect it to your FX client using Ankor.
  • Benefit from automatic model synchronisation.
  • No need for RPC like calls.
  • Changes to your model on server will automatically update your UI (changes get pushed).

HTML5

  • Develop web applications using your favorite HTML5 frameworks.
  • Bind your UI controls to server side view model using Ankor framework.
  • Ankor will synchronize your model. No need for RPC like calls.
  • Support for jQuery, Dojo, AngularJS.

ViewModel

  • Implement your View Model in Java.
  • Benefit from existing Java technologies (JEE, Spring, JPA).
  • Manage the state of your View Model (restore on login).
  • Support for JEE servers (Tomcat, GlassFish).
