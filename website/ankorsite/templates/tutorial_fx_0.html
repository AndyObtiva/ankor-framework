Before you start

Please check that all software components are installed properly. Especially check the Java FX section to make sure that JavaFX is on the Classpath.

  • Java
  • Maven
  • Java FX

JDK 1.7, download from here.

Make sure that

  • JAVA_HOME exists in your user variables (JDK installation directory)
  • and that %JAVA_HOME%\bin is in your Path environment variable

Open command line and test

java -version

Maven 3.0.5 or higher, download from here.

Make sure that

  • MAVEN_HOME exists in your user variables (JDK installation directory)
  • and that %MAVEN_HOME%\bin is in your Path environment variable

Open command line and test

mvn -version

To get JavaFX on the Classpath execute

mvn com.zenjava:javafx-maven-plugin:2.0:fix-classpath

On unix systems you may need sudo mvn ...

If you are interested in why this is needed read "Getting JavaFX on the Classpath"

Get the code

Clone git repository.

git clone https://github.com/ankor-io/ankor-phonecat

The folder ankor-phonecat is empty. To get the first tutorial step, checkout branch fx-step-0. This is how you may switch between tutorial steps later.

cd ankor-phonecat
git checkout -f fx-step-0

Now you got a maven project based on 3 modules

boot : Utility to start client and server
client : Client application (Java FX)
server : Server application

Hello World

Start application

cd boot
mvn install exec:java

This is what you get ...
